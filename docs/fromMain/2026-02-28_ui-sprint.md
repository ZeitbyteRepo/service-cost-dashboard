## Phase 2: Samsung Fold 6 Support

### Screen Modes

| Mode | Dimensions | Layout |
|------|------------|--------|
| **Cover Screen** | ~360px narrow | Single column, minimal, quick stats only |
| **Folded (portrait)** | ~740px | 2-column grid |
| **Unfolded (wide)** | ~1100px+ | 3-4 column grid |
| **Flex Mode** | Split screen | Top: dashboard, Bottom: selected provider details |

### Flex Mode Implementation

Use CSS `@media (vertical-viewport-segments: 2)` or JavaScript screen API:

```javascript
// Detect fold position
const isFlexMode = window.screen.isExtended ?? false;
// Or use Visual Viewport API
```

When in flex mode:
- **Top half:** Dashboard grid (compact)
- **Bottom half:** Selected provider details

### Cover Screen Support

When screen width < 400px:
- Show only: Total spend, provider count, health summary
- Minimal chrome, high contrast
- Touch-friendly tap targets
